MicroCMS Integration Area

This is where your blog content from MicroCMS will be displayed. Replace this placeholder with your MicroCMS embed code or API integration.

Integration Instructions:

Option 1: Direct Embed (Iframe)

If MicroCMS provides an iframe embed code, replace this entire placeholder div with:

<iframe src="YOUR_MICROCMS_URL" width="100%" height="800px" frameborder="0"></iframe>
Option 2: JavaScript SDK

Use MicroCMS JavaScript SDK to fetch and display content:

<div id="microcms-blog-container"></div> <script src="https://cdn.jsdelivr.net/npm/microcms-js-sdk@latest/dist/umd/microcms-js-sdk.js"></script> <script> const client = microcms.createClient({ serviceDomain: 'YOUR_SERVICE_DOMAIN', apiKey: 'YOUR_API_KEY' }); client.get({ endpoint: 'blogs' }).then(res => { // Display your blog posts here const container = document.getElementById('microcms-blog-container'); res.contents.forEach(post => { container.innerHTML += ` <article> <h2>${post.title}</h2> <p>${post.description}</p> </article> `; }); }); </script>
Option 3: Custom Integration

For more control, fetch content via MicroCMS API and render it with your custom HTML/CSS template.

Note: Make sure to secure your API keys and never expose them in client-side code for production sites. Consider using environment variables or server-side rendering.
